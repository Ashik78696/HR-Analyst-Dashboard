Project Overview
This project involves the creation of a comprehensive HR Analytics Dashboard designed to help organizations monitor workforce health. The primary focus is on analyzing Active Labors and identifying patterns in Employee Attrition based on various factors like age, department, and job satisfaction.

As an aspiring Data Analyst, I built this to demonstrate how data-driven insights can assist HR departments in improving retention and optimizing talent management.
